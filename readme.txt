Add Advanced Actors and Technicians support to WPMovieLibrary.

== Description ==

Add Actors and Technicians support to WPMovieLibrary. Actors and Technicians can have their own page with metadata of photo import, advanced sorting...

== Installation ==

1. Upload the plugin folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Use it!
